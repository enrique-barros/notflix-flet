Tema visual

Un único lugar donde viven todos los valores visuales de la app. Colores, tamaños de fuente, espaciados y radios de borde.

Por qué existe este archivo

Sin este archivo, cada componente usaría sus propios valores hardcodeados: "#E50914" en diez sitios distintos, 14 como tamaño de fuente en otros cinco. Si quisieras cambiar el rojo de Netflix, tendrías que buscar y reemplazar en cada archivo.

Con theme.py, cambias un valor aquí y se actualiza en toda la app automáticamente. Todos los demás archivos importan de aquí.

El código completo

config/theme.py
import flet as ft

# ── Paleta de colores ────────────────────────────
COLOR_PRIMARY  = "#E50914"   # rojo Netflix
COLOR_BG       = "#141414"   # negro de fondo
COLOR_SURFACE  = "#1F1F1F"   # superficie de cards
COLOR_SURFACE2 = "#2A2A2A"   # superficie elevada
COLOR_TEXT     = "#FFFFFF"
COLOR_SUBTEXT  = "#A3A3A3"   # texto secundario / apagado

COLOR_BADGE_BG = ft.Colors.with_opacity(0.18, ft.Colors.WHITE)
COLOR_CARD_BG  = "#1F1F1F"

# ── Tipografía ───────────────────────────────────
FONT_SIZE_LOGO       = 48
FONT_SIZE_SUBTITLE   = 18
FONT_SIZE_BODY       = 14
FONT_SIZE_BADGE      = 11
FONT_SIZE_CARD_TITLE = 13
FONT_SIZE_CARD_META  = 11

# ── Espaciado ────────────────────────────────────
PADDING_CARD  = ft.Padding(10, 10, 10, 10)
PADDING_BADGE = ft.Padding(10, 5, 10, 5)

# ── Radios de borde ──────────────────────────────
RADIUS_CARD  = ft.BorderRadius(6, 6, 6, 6)
RADIUS_BADGE = ft.BorderRadius(4, 4, 4, 4)

Cómo se usa en el resto del proyecto

Cualquier archivo que necesite un valor visual lo importa directamente:

ejemplo — cualquier archivo del proyecto
from config.theme import COLOR_BG, COLOR_TEXT, RADIUS_CARD

# en vez de escribir "#141414" a mano:
container = ft.Container(bgcolor=COLOR_BG)

ft.Colors.with_opacity()

Flet tiene su propio sistema para colores con transparencia. ft.Colors.with_opacity(0.18, ft.Colors.WHITE) crea blanco al 18% de opacidad. Se usa para fondos de badges y overlays semitransparentes.

ft.Padding() y ft.BorderRadius()

Ambos reciben cuatro valores: (izquierda, arriba, derecha, abajo) para Padding, y (superior_izq, superior_der, inferior_der, inferior_izq) para BorderRadius. En este proyecto todos los radios son iguales en las cuatro esquinas, dando bordes uniformemente redondeados.
