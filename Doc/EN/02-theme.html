Visual theme

A single place where all the app's visual values live. Colors, font sizes, spacing, and border radii.

Why this file exists

Without this file, each component would use its own hardcoded values: "#E50914" in ten different places, 14 as a font size in five others. If you wanted to change the Netflix red, you'd have to find and replace across every file.

With theme.py, you change one value here and it updates throughout the entire app automatically. All other files import from here.

The full code

config/theme.py
import flet as ft

# ── Color palette ────────────────────────────────
COLOR_PRIMARY  = "#E50914"   # Netflix red
COLOR_BG       = "#141414"   # background black
COLOR_SURFACE  = "#1F1F1F"   # card surface
COLOR_SURFACE2 = "#2A2A2A"   # elevated surface
COLOR_TEXT     = "#FFFFFF"
COLOR_SUBTEXT  = "#A3A3A3"   # secondary / muted text

COLOR_BADGE_BG = ft.Colors.with_opacity(0.18, ft.Colors.WHITE)
COLOR_CARD_BG  = "#1F1F1F"

# ── Typography ───────────────────────────────────
FONT_SIZE_LOGO       = 48
FONT_SIZE_SUBTITLE   = 18
FONT_SIZE_BODY       = 14
FONT_SIZE_BADGE      = 11
FONT_SIZE_CARD_TITLE = 13
FONT_SIZE_CARD_META  = 11

# ── Spacing ──────────────────────────────────────
PADDING_CARD  = ft.Padding(10, 10, 10, 10)
PADDING_BADGE = ft.Padding(10, 5, 10, 5)

# ── Border radii ─────────────────────────────────
RADIUS_CARD  = ft.BorderRadius(6, 6, 6, 6)
RADIUS_BADGE = ft.BorderRadius(4, 4, 4, 4)

How it's used across the project

Any file that needs a visual value imports it directly:

example — any file in the project
from config.theme import COLOR_BG, COLOR_TEXT, RADIUS_CARD

# instead of writing "#141414" by hand:
container = ft.Container(bgcolor=COLOR_BG)

ft.Colors.with_opacity()

Flet has its own system for colors with transparency. ft.Colors.with_opacity(0.18, ft.Colors.WHITE) creates white at 18% opacity. It's used for badge backgrounds and semi-transparent overlays.

ft.Padding() and ft.BorderRadius()

Both take four values: (left, top, right, bottom) for Padding, and (top_left, top_right, bottom_right, bottom_left) for BorderRadius. In this project all radii are equal on all four corners, giving uniformly rounded edges.
